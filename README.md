# Dark-Knight-Quiz-app
Dark knight trilogy quizz app
### Key tasks
* Designing the UI
* building the UI using react native
* Developping the quizz logic
* Saving the score data in localStorage

## Getting started

1. `npm install`

2. `npm start`
